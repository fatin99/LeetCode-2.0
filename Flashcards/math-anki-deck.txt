What is the greatest common divisor (GCD)?	The largest positive integer that divides two numbers without remainder.	formulas
Euclidean algorithm for GCD.	`gcd(a, b) = gcd(b, a % b)`; base case `gcd(a, 0) = a`.	formulas
Relationship between GCD and LCM.	`lcm(a, b) = a * b / gcd(a, b)`	formulas
What is a prime number?	A positive integer greater than 1 with exactly two divisors: 1 and itself.	formulas
Time complexity of the Sieve of Eratosthenes to find all primes up to n.	O(n log log n)	formulas
To test if n is prime, what's the smallest range of divisors you need to check?	2 up to √n — any composite must have a factor ≤ √n.	formulas
Modular arithmetic: what is `(a + b) mod m` in terms of `a mod m` and `b mod m`?	`((a mod m) + (b mod m)) mod m`	formulas
Fast exponentiation — how does it compute a^n in O(log n)?	If n is even: `pow(a, n) = pow(a*a, n/2)`. If odd: `pow(a, n) = a * pow(a, n-1)`. Halves the exponent each step.	techniques
Integer square root via binary search — what's the invariant?	Find the largest integer x such that `x*x <= n` by searching in [0, n].	techniques
Why might `0.1 + 0.2 == 0.3` evaluate to false in floating-point?	0.1, 0.2, 0.3 cannot all be exactly represented in binary floating-point — the sum carries a tiny rounding error.	techniques
Number of digits in a positive integer n (base 10).	`floor(log10(n)) + 1`	formulas
